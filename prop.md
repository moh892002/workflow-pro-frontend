الخصائص التقنية للمشروع:

1. التقنيات الأساسية:

- React 19.2.1 -
- TypeScript 5.8.2 -
- Vite 6.2.0 -
- React Router DOM 7.10.0 -
- Tailwind CSS -

2. (APIs):

- @google/genai 1.31.0 - واجهة برمجة تطبيقات Gemini للذكاء الاصطناعي
- Firebase 12.6.0 - للاستضافة والخدمات السحابية
- Recharts 3.5.1 - لمخططات البيانات
- Lucide React 0.555.0 - للأيقونات

3. البنية والمكونات:

- (Protected Routes) مع صلاحيات حسب الدور
- دعم ثنائي اللغة (العربية/الإنجليزية)
- (Dark/Light Mode)
- أنظمة Context API لإدارة:
  - (AuthContext)
  - (LanguageContext)
  - (ThemeContext)

4. الهيكل التنظيمي:

- (Components) - للعناصر القابلة لإعادة الاستخدام
- (Pages) - للشاشات الرئيسية (Dashboard, Tasks, Chat, etc.)
- (Services) - للعمليات المنطقية
- (Types) - تعريفات TypeScript

5. الصلاحيات والأدوار:

- ADMIN - صلاحيات كاملة
- HR_MANAGER - إدارة الموظفين والتقارير
- EMPLOYEE - صلاحيات محدودة

6. المميزات الرئيسية:

- نظام إدارة المهام والموظفين
- لوحة تحكم متكاملة
- نظام تقارير وماليات
- دردشة داخلية
- سلة المحذوفات (Recycle Bin)
- إدارة الملفات الشخصية

7. النشر والاستضافة:

- Firebase Hosting
- Vite (dist folder)
- (.env.local) للمفاتيح السرية
- دعم CDN للاستيراد الديناميكي

8. المظهر والتصميم:

- خطوط متعددة (Inter لللاتينية، Cairo للعربية)
- تخصيص ألوان مخصصة
- شريط تمرير مخصص
- حركات وتحولات سلسة

9. مراحل التشغيل:

- التطوير المحلي (npm run dev)
- البناء للإنتاج (npm run build)
- المعاينة (npm run preview)
